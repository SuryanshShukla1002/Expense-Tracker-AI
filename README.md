## API Reference

### **POST /api/expenses/add
-  **Description:** Adds a new expense entry for a user.

### **DELETE /api/expenses/delete/:id**
-  **Description:** Deletes an expense by ID.

### **GET /api/expenses/stats**
-  **Description:** Fetches expense statistics for the user.
